# 💻 My Dotfiles

A curated collection of my system configurations, shell setups, and automation scripts.

## 🚀 Quick Start

To apply these configurations to your machine:

```bash
git clone https://github.com/amadeu01/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x setup.sh
./setup.sh
```

## 📁 Structure

- `zsh/`: Zsh configuration split into modular files.
  - `exports.zsh`: Environment variables and path exports.
  - `plugins.zsh`: List of Oh My Zsh plugins.
- `packages.txt` & `yay_packages.txt`: Lists of system packages and AUR packages for easy reproduction.
- `docs/`: Documentation and changelogs.
- `setup.sh`: Installation script to symlink and configure the environment.

## 🛠️ Tech Stack

- **OS**: Arch Linux (Manjaro)
- **Shell**: Zsh + Oh My Zsh
- **Theme**: Bira
- **Package Manager**: Pacman & Yay
